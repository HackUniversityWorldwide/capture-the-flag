## 📘 About the Series
This workshop series is designed to introduce participants to beginner-friendly Capture The Flag challenges in a hands-on and accessible way.

CTFs are puzzle-based cybersecurity exercises where you solve challenges, follow clues, and find flags hidden in files, websites, code, or encoded text.

The goal of this series is to help participants build confidence with problem-solving, technical curiosity, and core beginner workflows used in entry-level CTFs.
